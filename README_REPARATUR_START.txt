- jarvis_3d_webgl

- INSTALL_DESKTOP_3D_ENGINE.bat

Installation:
1. Jarvis schließen.
2. ZIP entpacken.
3. Alles in deinen Jarvis-Hauptordner kopieren.
4. Ersetzen: Ja.
5. Zuerst starten: INSTALL_DESKTOP_3D_ENGINE.bat
6. Dann starten: SET_290GB_KI.bat
7. Danach starten: System einschalten

Wenn es wieder nicht startet:
JARVIS_DEBUG_START.bat starten und Screenshot vom Fehler senden.

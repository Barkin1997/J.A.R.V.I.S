JARVIS 3D FULLSCREEN + AUTO WAKEWORD + PROGRAMME MINIMIERT

Diese ZIP ist gemeinsam mit der 3D-Grafik.

Enthalten:
- app.py
- jarvis_3d_webgl
- INSTALL_DESKTOP_3D_ENGINE.bat
- System einschalten.bat

Funktion:
- startet als Vollbild über den ganzen Bildschirm
- echte WebGL 3D-Grafik
- kein Sprechen-Button
- hört automatisch auf: Jarvis, ...
- 3D-Core wird größer beim Zuhören/Antworten
- andere schwarze Startfenster/Dienste werden minimiert
- System einschalten.bat startet Zusatzdienste minimiert

Installation:
1. Jarvis schließen.
2. ZIP entpacken.
3. Alle Dateien/Ordner in den Jarvis-Hauptordner kopieren.
4. Ersetzen: Ja.
5. Starten mit: System einschalten

Vollbild schließen:
ALT + F4

JARVIS DIREKTMODUS + CODEX-PRO

Modell:
qwen3-coder-next:latest

Was neu ist:
- Du musst nicht mehr "Jarvis" sagen.
- Gesprochene Sätze werden direkt als Befehl verarbeitet.
- Code-/Datei-Aufgaben gehen automatisch an lokalen Codex-Pro.
- 3D wird NICHT geändert.
- Diese ZIP enthält KEINEN jarvis_3d_webgl Ordner.

Beispiele, einfach sagen:
- codex status
- prüfe dich
- finde warum du nicht antwortest
- repariere app.py
- suche OLLAMA_MODEL
- lese brain.py
- test
- öffne Chrome
- mach Screenshot

Sicherheit:
- Codex-Pro macht Backups vor Dateiänderungen.
- 3D bleibt geschützt.
- Geschützte Ordner wie .venv, external, models werden nicht geändert.

Installation:
1. Jarvis schließen.
2. ZIP entpacken.
3. Inhalt in deinen Jarvis-Hauptordner kopieren.
4. Ersetzen: Ja.
5. Einmal starten:
   DIREKTMODUS_AN.bat
6. Danach:
   System einschalten

Wenn du später wieder "Jarvis" als Aktivierungswort willst:

BUTTON-NAME: System einschalten

So machst du den Button:

1. ZIP entpacken.
2. Alles in deinen Haupt-Jarvis-Ordner ziehen.
   Das ist der Ordner, wo "Starte Jarvis KI.bat" liegt.
3. Doppelklick auf:
   DESKTOP_BUTTON_SYSTEM_EINSCHALTEN_ERSTELLEN.bat

Dann erscheint auf dem Desktop:

System einschalten

Dieser Button startet:
- Ollama
- ComfyUI
- ComfyUI im Browser
- Jarvis

Wichtig:
Wenn du KI-Bilder oder KI-Videos machen willst, muss das schwarze ComfyUI-Fenster offen bleiben.
